## 📌 Project Overview
This is a PHP-based Image Gallery web application that allows users to register, log in, upload images, and view them in a dynamic gallery. The project demonstrates secure authentication, file handling, and MySQL database integration.

---

## 🚀 Live Demo
http://final-exam.infinityfree.me/login.php

---

## ✨ Features

- User Registration system
- Secure Login & Logout (Session-based authentication)
- Password hashing using `password_hash()`
- Image upload with validation
- Dynamic image gallery display
- Delete image functionality
- MySQL database integration (PDO)
- Responsive UI using CSS

## 📸 Functional Flow

1. User registers an account
2. User logs in securely
3. User uploads images
4. Images are stored in `/uploads` folder
5. Image details are saved in MySQL database
6. Dashboard displays all uploaded images
7. Users can delete images
